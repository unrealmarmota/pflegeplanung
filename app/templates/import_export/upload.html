Dienstplanwünsche importieren

XLSX-Datei mit Mitarbeiter-Präferenzen hochladen

Datei hochladen
Erwartet das Format "Dienstplanwünsche MA" mit Spalten: Name, Dienste (F/KD/T/S/N), Nächte, Präferenzen, etc.
Was wird importiert?
  • Dienst-Präferenzen: Welche Schichten bevorzugt werden (F/KD/T/S/N)
  • Nacht-Konfiguration: Anzahl/Block-Länge der Nachtdienste
  • WE-Limits: Max Wochenenden pro Monat
  • Freitext-Wünsche: Werden angezeigt, müssen ggf. manuell eingepflegt werden
Bestehende Präferenzen werden beim Import überschrieben. Regel-Ausnahmen werden ergänzt (nicht gelöscht).
